# Alcohol Detection Seat Belt System

This project is designed to improve road safety by detecting alcohol levels in the driver. 
An alcohol sensor checks the driver’s breath and if alcohol is detected above the limit, 
the vehicle engine will not start.

The system also checks whether the driver is wearing a seat belt. If the seat belt is not 
fastened, the engine will remain locked. The project can also send emergency alerts and 
monitor vehicle speed.

## Features
- Alcohol detection using sensor
- Engine lock if alcohol is detected
- Seat belt safety detection
- Speed monitoring
- Emergency alert system

## Components Used
- Microcontroller (Arduino)
- Alcohol Sensor (MQ-3)
- Seat Belt Sensor
- GSM Module
- Buzzer and LEDs
